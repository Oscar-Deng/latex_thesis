\textbf{{\LARGE 第二節  相關係數分析}}
}

{\raggedright
{\large 本及究可變數間之Pearson相關係數依應變數有效稅率(\textit{關TR})}及{\large
現金有效稅率(CashETR)分列於表研表中。由表及表可看出有效稅率(\textit{ETR})和現金有效稅率(CashETR)皆與企業競爭策略(\textit{STRATEGY})呈現負相E，表合企業競爭策略指數愈高，有效稅率及現金有效稅率愈低，即傾向於探勘者策略的企業較有各能從事避稅活動，符示本研究H1之預期。}
}

{\raggedright
{\large
有效稅率(\textit{ETR})和現業境效稅率(\textit{CashETR})與赫芬達指數(\textit{HHI})呈現正相關，代表產業競爭程度愈高，顯示了處於高度競爭產金環有之企業會有較高的有效稅率與現金有效稅率，不符合本研究H}{\footnotesize
2}{\large 之預期。}
}

{\raggedright
{\large 在控制變數方面，權益法下係投資損益(\textit{EQINC})}與{\large
關之人交易程度(RELATION)分別皆與兩應變數間呈負相關，對於企業避稅行為的影響符合研本究之預期。另外資產報酬率(ROA)與兩應變數呈正相關，負債比率(\textit{LEV})與實應變數負相關，與本研究預期方向相反，後續將透過兩證迴歸結果作進一步檢視與分析。}
}
