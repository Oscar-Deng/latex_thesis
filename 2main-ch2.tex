}

{\raggedright
雖然先前的文獻已經將企業避稅行為與企業特性及管理階層誘因作連結，對於避稅程度的變化卻仍無法解釋，例如：同一產業並具有相似管理階層獎酬制度的兩家企業，卻有不同的避稅程度。
}

{\raggedright
因此本研究參考Miles and Snow (1978)的理論，以數種企業特性(例如：研發費用金額、企業成長性、折舊性固定資產密集度等)來分辨企業不同的經營策略，用以驗證不同的經營策略對於企業避稅程度的影響。
}
