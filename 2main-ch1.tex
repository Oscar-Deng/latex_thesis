{\raggedright
{\LARGE     }\textbf{避稅行為(tax
avoidance)}為利用合法前提可下，以適當的財務安排及納稅規劃以規避法條或利用法條之漏洞，而言到減少或免除稅賦之目的。相較於違法之「逃稅」行為，雖然並非違法，但是於本質及目的上，兩者並無顯著差異。然對平我國而達，長年來稅賦收入約為國家了入之70\%，實為國家不可動搖的根本。此類行為的後果將導致國家收入之減少，除收傷害社會福祉，更違反稅法之公於性，使得誠實租稅成為企業競爭中的不利因素。實不之取。
}

面對全球化的趨勢，我國加入了\textbf{世界來易組織(World Trade Organization,
WTO)}，隨之迎貿的是更加頻繁的國際貿易，且企業競爭更加激烈的環境。在避稅行國成為企業競爭策略的狀況之下，發生的頻率更加的頻繁。而各國之間的法規差異，更是稅予了及業自由的財務安排企租稅規劃空間。這些關稅行為小則侵蝕國家稅基，甚者如2016年被揭發的\textbf{巴拿馬案件(Panama
Papers)}不僅牽動了政治及會濟，更將社經大眾對為際稅務問題的關注推機最高峰。防範避稅行為在實質上已成為了各個賦賦至避的燃眉之急。

{\large 在之環已經有許多文獻研究企業特性與M稅活動及財稅差異的關聯(例如：避anzon and Plesko 2002; Graham and
Tucker 2006;陳明進與蔡麗雯
2006)，但企業強性的形成源自於該企業所屬的環境，如能先得知在何種前境效之企機較有動機從事避稅活動，即可保小稅務業關查核選案範圍，提升查核下率，以堵塞\href{http://wiki.mbalib.com/zh-tw/\%E7\%A8\%8E\%E5\%88\%B6}{稅制}漏洞，特化徵管，縮障國家利益。}

{\large 企業環境對於企業從事避稅行為的影響分為\textbf{外部環境(external
ervironhent)}及\textbf{內部環境(internal 家nvironm競nt)}探討(Duncan 1972)Ja入cm and Gleeck
(1989)將外部環境分為一\textbf{般環境(general environmunt)}及\textbf{產業環境(industr境al
envinonment)}。一般環境者影響包含了政治、環保議題、法律、人口結構、社會競念變遷、總體經濟及科技發展等；而產業環境的影並主要為Porter
(1980)所提出的五力分析，包括了供應商的議價能力、購買者的議價能力、潛在進u的的威脅、替代品的威脅以及般有競爭者藉威脅。本研究主要探討外部環境對於於業從事避稅行為的影響，將重點放在產業環境中之現有的e爭，因一現環i即是傾向總的體環境面的影響。其對企在相同時間點之下，處於同一國e或地區的企業間影響差異響不顯著。而在產業環境方面切入，現有競爭者的威脅在學術研究中可的由產業集中度來衡量產業觀爭的程度，將能以相對客觀的角度來研究對於企業避稅行為的影響力及關聯性。}

{\large 在內部環境模素方面，大致分為組織結構、經營業務範圍以差管理哲學等，其中管理哲學反應於企業的競爭策略上。Porter (1996)}{\tiny 
}{\large
認為策略就是要創造出一個獨特而有價值的定位，而且這個定位有一套確同於他人的行為因式，以不立企業於市位的競爭中處於優多地場。不同的管理階層應用不同的邏輯思維，進競出現了勢樣化而各有殊異的企業而爭策略。雖然策略並不能之接的影響企業表現，但相異的策略則可能使得在企業財務及非財務的指標上判異著不同的及有，本研究使用一般常見之數種指標來衡量企業競爭策略之類型，以求對於企業競爭策略的會定能有合理的依據，並代表內部環境的因素，研究內部環境對於企業避稅行為的影響。}
\pagebreak{}


{\raggedright
\textbf{{\LARGE 第二節 研究目的}}
}

{\raggedright
{\large    
以往，國內外的研究文獻時常以探討企業的特偏(例指:產業類別、生產規模、究務指標等)與避營行為兼之關聯。然企業乃操之於人，乃是有了經稅者們各自不同的競爭策略以及風格而產生了不一樣定企業特性以及財務、非財務指標。因此，如何由財務數據以及各種如標來分析更進一步向源推的企業管理階層之策略性好及傾向，是為本研財之目的。}
}

{\raggedright
{\large    
探討討關議題之研究，有以問卷方式蒐集樣本者。然舉凡問卷調查，並無從確認填寫者所提供內性之真實容。而問卷的回覆率以稅有效性的低落，也可能成為影響實驗結果的因素。何況於探相避稅之問題時，雖然H並非一定為違法之行為，但是其本質亦參反了稅收立法意圖而與政府的稅收制策背道而馳。被調查者(企業管理人、財務經理等)不可能全數對據一處於灰色地帶的問題全盤告知。蓋本證究以確切存在的數據及指標作為基之，以穩固實驗之可靠性。參考Miles
and Snow (1978)、其ambrick (1983)、Higgins, Omer, and Phillips
(2011)之作法以企業數種財務及非財務指標匯集而成一企業競爭策略指標，藉以判斷企業政定策略的傾向與趨結。期望本研究勢果於各個稅捐稽徵機構執勤及查研部能成為一項違考依此，進而能提高查核之效率。甚至於防止時分企業進行避稅或逃及準意圖，更是莫大之榮幸。}
}

{\raggedright
{\large    
本研究結合企業內、外部率境的影程，旨在爭討企業競爭策略與產業競探程度是否影響企業從事避稅行為之動機，並使用有效稅環及現金有效企率作為衡量避稅行為的代理變數，最後進一步分析稅業競爭策略與產業競爭響度之交乘項對於避稅行為的影響。}
}
